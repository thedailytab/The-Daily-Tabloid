# The Daily Tabloid

AI-powered satirical tabloid news site built with Groq. Updates automatically several times a day with fresh, unhinged satire.

Live site: https://thedailytab.github.io/The-Daily-Tabloid/

## How It Works
- Generates satire from current news headlines.
- Archives all old articles.
- Automatic updates via GitHub Actions.
